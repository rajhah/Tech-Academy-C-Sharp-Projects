# Tech-Academy-C-Sharp-Projects

This repository contains all the projects from the C# course of the Tech Academy bootcamp

## Projects
- [CarInsurance](#CarInsurance)
- [CodeFirstDB](CodeFirstDB)
- [NewsletterSignupDemo](NewsletterSignupDemo)
- [TwentyOne](TwentyOne)

## Demos
Everything else in this folder is some sort of assignment designed to demonstrate use of various object-oriented concepts.

## CarInsurance
This is a .NET Framework web application with an MVC model. It has full CRUD functionality with the associated database. It will take user input on a form page and determine an insurance quote based on several different factors.

## CodeFirstDB
This project is a demonstration of using a code-first model to create a database in a .NET console app. Users can add to and view members of a sample database.

## NewsletterSignupDemo
This project is .NET Core web application with an MVC model. It implements ViewModels as well as database access via Entity Framework.

## TwentyOne
This project is a console app version of the classic card game Blackjack. It covers many object-oriented programming concepts such as classes, class inheritance, and abstract and static classes. It uses several types of data structures extensively, e.g. lists, enums, and dictionaries, and also implements some rudimentary exception handling.
